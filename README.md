# 🌱 KrishiDoot

KrishiDoot is an AI-powered smart agriculture platform built to help farmers make better farming decisions using machine learning, weather-based crop recommendations, fertilizer guidance, and a digital marketplace.

---

## 🚀 Live Features

✅ Crop Recommendation using Soil + Weather Inputs  
✅ Fertilizer Advisor for 22 Crops  
✅ Farmer Marketplace  
✅ User Authentication (Firebase Login / Signup)  
✅ Dashboard  
✅ Prediction History Stored in Firestore  
✅ Responsive Modern UI

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Flask (Python)
- Machine Learning Models
- REST API

### Database / Auth
- Firebase Authentication
- Firebase Firestore

### Deployment
- Frontend: Vercel
- Backend: Render
